Installation: Famous 5-minute install

  1. Unzip the package in an empty directory and upload everything.
  2. Open wp-admin/install.php in your browser. It will take you through the process to set up a wp-config.php file with your database connection details.
    1. If this does not work, open wp-config-sample.php with a text editor and fill in your database connection details.
    2. Save the file as wp-config.php and upload it.
    3. Open wp-admin/install.php in your browser.
  3. Once the configuration file is set up, the installer will create the required database tables. If an error occurs, double-check your wp-config.php file.
  4. If you did not enter a password, note the generated password. If no username was provided, it will be admin.
  5. The installer will then direct you to the login page. Sign in and update your password if required.

Updating

Using the Automatic Updater

  1. Open wp-admin/update-core.php in your browser and follow the instructions.
  2. That’s it!

Updating Manually

  1. Back up any files you may have modified.
  2. Delete old WordPress files, keeping custom files.
  3. Upload the new WordPress files.
  4. Visit /wp-admin/upgrade.php.

Migrating from Other Systems

WordPress can import content from many platforms . Install WordPress first, then use the import tools.

System Requirements

  • PHP version 7.4 or greater.
  • MySQL version 5.5.5 or greater.

Recommendations

  • PHP version 8.3 or greater.
  • MySQL version 8.0 or greater, or MariaDB version 10.6 or greater.
  • The mod_rewrite Apache module.
  • HTTPS support.
  • A link back to wordpress.org.
