Installation: Famous 5-minute install

  1. Unzip the package in an empty directory and upload everything.
  2. Open wp-admin/install.php in your browser. It will take you through the setup process for wp-config.php.
    1. If it fails, open wp-config-sample.php in a text editor and fill in your database details.
    2. Save it as wp-config.php and upload.
    3. Reopen wp-admin/install.php.
  3. The installer sets up your database. If it fails, recheck your wp-config.php.
  4. Note down the auto-generated password if you didn’t enter one. Default username is admin.
  5. Login at wp-login.php. Update password via “Profile” if needed.

Updating

Automatic Update

  1. Visit wp-admin/update-core.php and follow the instructions.
  2. Done!

Manual Update

  1. Backup modified files (like index.php).
  2. Delete old files (except modified ones).
  3. Upload new files.
  4. Visit wp-admin/upgrade.php in your browser.

Migrating from Other Systems

Use import tools after installation. Access them via wp-admin/import.php.

System Requirements

  • PHP version 7.2.24 or greater
  • MySQL version 5.5.5 or greater

Recommended

  • PHP 8.3+
  • MySQL 8.0+ or MariaDB 10.6+
  • mod_rewrite Apache module
  • HTTPS support
  • Link back to wordpress.org
